Class - bu HTML elementlariga bir yoki bir nechta xususiyatlarni birlashtirish imkonini beruvchi atributdir. CSS da class lar orqali bir nechta elementlarga bir xil uslub berish mumkin.

Id - bu HTML elementini noyob ravishda aniqlash uchun ishlatiladigan atributdir. Har bir id faqat bitta elementga berilishi kerak va CSS da id orqali aniq bir elementga uslub berish mumkin.

BEM (Block Element Modifier) - bu CSS uslublarini tashkil qilish va nomlash uchun ishlatiladigan metodologiya. BEM yordamida kodni o'qish va tushunishni osonlashtirish mumkin. BEM da har bir element blok, element va modifikatorlardan iborat bo'ladi.

Class berish qoidalari quyidagilardan iborat:

  • Class nomi kichik harflar bilan boshlanishi va bo'sh joylar o'rniga chiziqchalar ( - ) ishlatilishi kerak.
  • Class nomi ma'noli va tushunarli bo'lishi lozim.
  • Bir elementga bir nechta class berish mumkin.

Class va Id o'rtasidagi farqlar:

  • Class bir nechta elementlarga berilishi mumkin, Id esa faqat bitta elementga beriladi.
  • Class nomi takrorlanishi mumkin, Id nomi esa noyob bo'lishi kerak.
